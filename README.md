# 项目介绍
  创建一个后台管理系统的首页
  项目基于vue3 + element-plus 实现
  项目采用pinia 状态管理
  项目采用vue-router 路由管理

# 界面介绍
  ## header 
    项目采用element-plus 提供的header 组件
    左侧 logo 图片和项目名称（智慧运维）  
    右侧 切换主题、 登录按钮
  ## 左侧导航菜单 
    项目采用element-plus 提供的menu 组件
    导航菜单包含 首页、 用户管理 模块
    点击模块可以跳转到对应的页面,每个页面是一个单独的页面文件，页面文件的路径与路由路径一致
    每个页面包含一个标题即可，标题与模块名称一致
    页面文件统一放在 src/views 目录下
  ## 内容区域
    项目采用element-plus 提供的main 组件
    内容区域包含当前选中的模块内容
